# Мой школьный день

**Мой школьный день** — Android-приложение для ведения школьного расписания и списка вещей, которые нужно взять на конкретный урок в выбранный день.

Приложение позволяет редактировать расписание по дням недели, а также создавать отдельные списки предметов/вещей для каждого урока. Данные сохраняются локально на устройстве и остаются доступными после перезапуска приложения.

---

## Возможности

- просмотр расписания по дням недели;
- редактирование списка уроков для каждого дня;
- добавление и удаление предметов в расписании;
- отдельная вкладка со списками “что взять с собой”;
- выбор дня недели для просмотра сборов;
- добавление и удаление вещей для конкретного урока;
- локальное сохранение данных через DataStore;
- восстановление расписания и списков после перезапуска приложения;
- современный минималистичный интерфейс на Jetpack Compose и Material 3.

## Основные экраны

### Расписание

На главном экране отображаются дни недели в виде карточек.

При выборе дня открывается редактор, где можно изменить количество уроков и названия предметов.

### Сборы

На экране сборов можно выбрать день недели и добавить список вещей для каждого урока.

Например: учебник, тетрадь, ручка, форма, дневник и другие необходимые предметы.
